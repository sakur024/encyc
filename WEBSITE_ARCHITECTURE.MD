# Implementation Rules

* Knowledge First
* Recommendation Second
* Every page must educate.
* Every page must contain diagrams or visual learning.
* Every technical term must be explained.
* Every specification must have a beginner-friendly explanation.
* Every article must connect to other articles.
* No isolated pages.
* Consistent page structure across all devices.
* Responsive by default.
* Accessible (WCAG AA).
* Fast loading.
* SEO optimized.
* Scalable for thousands of articles.

---

# Final Goal

Build the world's most comprehensive technology knowledge platform where users can **Learn → Understand → Explore → Compare → Decide**, using structured educational content instead of product-focused pages.
